What's GWeb?


Spreadsheet(Excel)을 database와 business logic server로 사용하여 UI 소스 만으로 웹앱을 만들수 있도록 지원하는 라이브러리입니다.

- Library

1. Server-side Library for Google Script Api (gweb-js_s.js)
Google Script Api를 이용하여 Google Spreadsheet을 쉽게 관리할 수 있도록 도와 줍니다.
git clone https://doohee323@bitbucket.org/doohee323/gweb-js.git
2. Client-side Library
Google OAuth2를 통해 Server-side Library와 통신할 수 있도록 지원합니다. 또한 Parameter와 Resultset의 처리를 지원합니다.
3. Client-side Java Library
안드로이드 또는 기존 Web application에서 활용하기 위해서 Java library를 이용할 수 있습니다.
git clone https://doohee323@bitbucket.org/doohee323/gweb-jsc.git

- Usage

1. Excel 파일의 재사용
MS Excel 파일의 sheet와 함수들을 그대로 Database와 Business logic으로 활용할 수 있습니다.
2. Backend 서버없이 웹앱 개발
Backend에 Web application server 또는 별도의 Language(ex. java, pyton, php etc) 없이 Spreadsheet의 function을 이용해서 웹앱을 만들 수 있습니다.

다양한 usecase에 필요한 library와 example을 제공함으로써 UI소스만으로 웹개발을 할 수 있도록 지원합니다.
